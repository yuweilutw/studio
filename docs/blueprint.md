# **App Name**: WanderWise

## Core Features:

- Input Capture: Capture user inputs for location, duration, travel theme, age range, and preferences.
- Personalized Recommendation Engine: Generates travel recommendations based on the captured inputs.
- Travel Checklist: Create a travel checklist with key things to remember depending on the inputs and the weather in the chosen location. The checklist generation uses an AI tool to dynamically include the right kinds of items based on its reasoning about the various inputs.
- Weather & Attire: Provide weather forecasts and appropriate clothing suggestions for the travel destination.
- Transportation Tips: Offer insights into local transportation options and advice.
- Estimated Cost Range: Provide an estimated cost range for the trip based on destination and duration.
- Image Integration: Display relevant images sourced from Google Images to enhance the user's understanding of the location.

## Style Guidelines:

- Primary color: A gentle sky blue (#87CEEB) to evoke a sense of calm and exploration.
- Background color: Very light blue (#F0F8FF), creating a clean and airy feel.
- Accent color: Muted yellow (#F0E68C) to highlight important information and call-to-action buttons, conveying a sense of warmth.
- Font pairing: 'Poppins' (sans-serif) for headlines, lending a geometric, fashionable look, paired with 'PT Sans' (sans-serif) for body text to ensure readability.
- Use clean, minimalist icons to represent various categories and suggestions.
- Implement a card-based layout to organize and present information in a digestible format.
- Incorporate subtle transitions and animations to enhance user engagement without overwhelming the interface.
